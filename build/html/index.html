class module_4.src.order.Order

This is a class to generate an Order object

input_pizza(crust, sauce, cheese, toppings)

Creates a pizza object including the crust, sauce, cheese, toppings, and cost of the pizza. Appends the pizza to the order, and updates the total cost of the order.

Parameters:

  • crust – crust type including thin, thick, or gluten free

  • sauce – sauce type including marinara, pesto, or Liv Sauce

  • cheese – cheese type including mozerella only

  • toppings – list of toppings including pineapple, pepperoni, or mushrooms
